Gowin FPGA Labs

Documentación práctica para compilar, cargar y estudiar ejemplos en Verilog y VHDL sin amarrar el SDK a una tarjeta específica.

Instalación Ver ejemplos

Funciona con las siguientes herramientas

Flujo portable

Usa DevLab como entrada principal para evitar depender de Make en Windows o macOS.

Dos HDLs

Cada práctica conserva Verilog como flujo por defecto y VHDL como variante explícita.

Target configurable

La referencia actual usa un GW1NR-LV9QN88PC6/I5, pero la tarjeta se define por los archivos CST y la sección flash.

Soporte probado

Flujo documentado y probado en Windows 11 y Ubuntu 24.04.
