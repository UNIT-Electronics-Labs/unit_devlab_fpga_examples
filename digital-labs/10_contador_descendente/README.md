# Practica 10: Contador descendente

Compilar con DevLab:

```bash
cd digital-labs/10_contador_descendente
devlab build
devlab flash
```

Para probar VHDL explicitamente:

```bash
cd digital-labs/10_contador_descendente
devlab build -c devlab-vhdl.toml
```

El `Makefile` queda como compatibilidad para Linux:

```bash
make -C digital-labs/10_contador_descendente
```
