# Practica 09: Contador ascendente

Compilar con DevLab:

```bash
cd digital-labs/09_contador_ascendente
devlab build
devlab flash
```

Para probar VHDL explicitamente:

```bash
cd digital-labs/09_contador_ascendente
devlab build -c devlab-vhdl.toml
```

El `Makefile` queda como compatibilidad para Linux:

```bash
make -C digital-labs/09_contador_ascendente
```
